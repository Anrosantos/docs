</Note>

## Create a webhook
Command:`;webhook create` <br />
Description: Create a new webhook. <br />

<Tabs>
    <Tab title="Example">
        <img src="/images/webhooks/wc.webp"/>
    </Tab>
    <Tab title="Syntax">
       ```
       ;webhook create [channel] [name]
       Example: ;webhook create #general rules!
       ```
    </Tab>
</Tabs>

<Tip>
    You may want to copy the indentifier if you're going to utilize the webhook right afterwards. If lost, use `;webhook list` to retrieve it.
</Tip>



## Sending messages with webhooks
Command:`;webhook forward` <br />
Description: Forward a message through a webhook. <br />

<Tabs>
    <Tab title="Example">
        <img src="/images/webhooks/wf.png"/>
        ```
        ;webhook forward X7W5o7EB Hello, how are you?
        ```
    </Tab>
    <Tab title="Syntax">
       ```
       ;webhook forward (identifier) (script or message)
       Example: ;webhook forward X7W5o7EB Hello, how are you?
       ```
    </Tab>
</Tabs>



## Editing webhook messages
Command:`;webhook edit` <br />
Description: Edit a message sent by a webhook. <br />

<Tabs>
    <Tab title="Input">
     <img src="/images/webhooks/we.png"/>
    </Tab>
    <Tab title="Output">
        <img src="/images/webhooks/out.png"/>
    </Tab>
    <Tab title="Syntax">
    ```
    Syntax: ;webhook edit (message id) (script or message)
    Example: ;webhook edit https://discord.com/channels/1331173093124608062/1331173093124608065/1336835116122509354 I'm good
    ```
    </Tab>
</Tabs>
<Warning>
    This doesn't customize the name, profile, or how the webhook looks whatsoever, rather a message it sent.
</Warning>
