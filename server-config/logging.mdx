Subcommand: `logging enable` <br /> Module: `logging` <br /> Description: Set events to be logged in a channel.

<div id="enable"/>
<Tabs>
  <Tab title="Example">

    </Frame>
  </Tab>
  <Tab title="Syntax">`logging enable <channel> [event]`</Tab>

</Tabs>

Subcommand: `logging ignore` <br /> Module: `logging` <br /> Description: Ignore a channel or user from being unintentionally logged.

<div id="ignore"/>
<Tabs>
  <Tab title="Example">
    `logging ignore @x`
    <Frame caption="Ignoring @x from all logs">
      <img src="/images/security/logging/logging_ignore.gif" />
    </Frame>
  </Tab>
  <Tab title="Syntax">`logging ignore <member|channel>`</Tab>
    <Tab title="Functionality">
    ```mermaid
  flowchart LR
    subgraph Ignored?
        direction TB
        top1[Yes] --> bottom1[Do not log from this user/channel and <br /> do not save in database]
        top2[No] --> bottom2[Keep logging from this user/channel and <br /> save in database]
    end

    Event --> Ignored?
```

  </Tab>
  <Tab title="Punishment Types">
    No flags for punishment are avaliable for this command.
  </Tab>
</Tabs>
