Reward users for representing your server vanity in their status.

Subcommand: `vanity channel` <br /> Module: `config` <br /> Description: Set the channel that vanity messages are sent to.

<div id="channel"/>
<Tabs>
  <Tab title="Example">
  `vanity channel #general`
  <Frame caption="Enabling vanity log channel.">
  <img src="/images/vanity/vanity_channel.gif" />
  </Frame>
  </Tab>
  <Tab title="Syntax">`vanity channel`</Tab>
</Tabs>

<br />

Subcommand: `vanity channel remove` <br /> Module: `config` <br /> Description: Remove the channel that vanity messages are sent to.

<div id="channel"/>
<Tabs>
  <Tab title="Example">
  `vanity channel #general`
  <Frame caption="Disabling vanity log channel.">
  <img src="/images/vanity/vanity_channel_disable.gif" />
  </Frame>
  </Tab>
  <Tab title="Syntax">`vanity channel`</Tab>
</Tabs>

<br />

<br />

Subcommand: `vanity role` <br /> Module: `config` <br /> Description: Change or set the role that is given upon adverising.
