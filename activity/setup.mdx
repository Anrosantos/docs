# Setting Up Activity Integration

Setting up the Evict Activity integration is a straightforward process. Follow these steps to get started with displaying your music status on Discord.

## Installation Steps

### 1. Invite the Integration

Copy and paste the following invite link into any Discord channel:

```bash
https://discord.com/oauth2/authorize?client_id=1323720110787268768
```


### 2. Add to Discord

After posting the invite link:
1. Click on the link in your Discord channel
2. Select "Add App" in the authorization window
3. Follow any additional Discord prompts to complete the installation

### 3. Enable Activity

There are two ways to enable the Activity integration:

#### Option 1: Activity Shelf
- Open Discord's Activity shelf
- Locate Evict Music Activity
- Click to enable

#### Option 2: Through Music Commands
When using the `;play` command, you'll receive a prompt asking if you'd like to enable the Activity integration:
- Respond to the prompt to enable the integration
- Your music status will begin displaying automatically

## Activity Interface
The following is the interface for the Activity integration:

- Shows currently playing song
- Shows current queue
- Shows current volume
- Shows current loop status
- Shows current track duration
