---
title: "Distort Commands"

icon: "shuffle"
---

<Tip>
  All distort commands require an image attachment.
</Tip>

## Geometric Distortions

<CardGroup cols={2}>
  <Card title="Pyramid" icon="pyramid">
    `distort pyramid` (*attachment) - Apply a pyramid-like distortion effect
  </Card>
  <Card title="Shear" icon="arrows-left-right">
    `distort shear` (*attachment) - Apply a shearing distortion effect
  </Card>
  <Card title="Stretch" icon="arrows-up-down">
    `distort stretch` (*attachment) - Apply a stretching distortion effect
  </Card>
  <Card title="Poly" icon="octagon">
    `distort poly` (*attachment) - Create a polygonal distortion pattern
  </Card>
</CardGroup>

## Destructive Effects

<CardGroup cols={2}>
  <Card title="Shred" icon="scissors">
    `distort shred` (*attachment) - Shred the image into strips
  </Card>
  <Card title="Slice" icon="knife">
    `distort slice` (*attachment) - Slice the image into segments
  </Card>
  <Card title="Burn" icon="fire">
    `distort burn` (*attachment) - Apply a burning distortion effect
  </Card>
  <Card title="Melt" icon="droplet">
    `distort melt` (*attachment) - Make the image appear to melt
  </Card>
</CardGroup>

## Motion Effects

<CardGroup cols={2}>
  <Card title="Phase" icon="wave-square">
    `distort phase` (*attachment) - Apply a phasing distortion effect
  </Card>
  <Card title="Dizzy" icon="hurricane">
    `distort dizzy` (*attachment) - Create a dizzying spiral distortion effect
  </Card>
  <Card title="Infinity" icon="infinity">
    `distort infinity` (*attachment) - Apply an infinity mirror effect
  </Card>
  <Card title="Endless" icon="rotate">
    `distort endless` (*attachment) - Create an endless looping distortion
  </Card>
</CardGroup>
